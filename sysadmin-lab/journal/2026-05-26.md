# 2026-05-26 - Linux Essentials aplicado al laboratorio

## Practica

Diagnostico basico en Linux usando el proyecto IntraCorp Helpdesk como escenario real.

## Objetivo

Continuar la racha de aprendizaje sin forzar memorizacion. La idea fue crear una practica pequena para reconocer comandos utiles de Linux y conectarlos con tareas SysAdmin.

## Comandos foco

```text

```

## Aprendizaje principal

No hace falta recordar todos los comandos de memoria en un solo dia.

Para Linux Essentials conviene pensar en preguntas:

```text
Donde estoy?
Que archivos hay?
Que dice este archivo?
Que paso en los logs?
Donde aparece esta palabra?
La maquina tiene recursos?
La app responde?
Docker ve los servicios arriba?
```

Luego cada comando responde una pregunta concreta.

## Conexion con IntraCorp Helpdesk

La practica usa el laboratorio para revisar:

- Estructura del proyecto.
- Logs del backend.
- Health checks HTTP.
- Estado de contenedores.
- Conectividad entre servicios.
- Recursos basicos del sistema.
